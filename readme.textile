h1. About

Examples of "Mongo Ruby Driver":http://api.mongodb.org, "Mongoid":http://mongoid.org/ and "MongoMapper":http://mongomapper.com/ running on "MongoHQ":http://mongohq.com.

h2. Requirements

You need a "MongoHQ":http://mongohq.com account and should create a "Gemset with RVM":http://rvm.beginrescueend.com/gemsets/creating/.

h2. Gems via RVM

<pre>
# use Ruby 1.9 with a gemset
rvm use 1.9.1@mongodb
# import the gems from provided .gems file
rvm gemset import
</pre>

h2. Execution

<pre>
export MONGOHQ_USER="username"
export MONGOHQ_PASS="password"
export MONGOHQ_PORT="port"
export MONGOHQ_DB="database"

rake
</pre>
